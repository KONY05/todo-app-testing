# Todo App

A Vite-powered React + TypeScript todo app with:

- add, complete, and delete tasks
- all / active / completed filters
- clear completed
- local storage persistence
- responsive styling with light and dark mode support

## Run it

Install dependencies:

```sh
npm install
```

Start the development server:

```sh
npm run dev
```

Then visit the URL printed by Vite, usually:

```text
http://localhost:5173
```

## Build

```sh
npm run build
```
